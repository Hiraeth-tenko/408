# 栈

---

## 栈定义

栈顶：允许操作的一侧。

栈底：固定，不允许操作的一侧。

特点：

- 只允许在一侧进行操作
- 后进先出

## 基础操作

- InitStack (&S)：初始化一个栈
- Empty (&S)：栈 S 是否为空
- Push (&S, x)：将元素 x 加入到栈 S
- Pop (&S)：弹出顶部元素
- Top (&S)：获取栈顶元素
- DestoryStack (&S)：销毁栈 S
