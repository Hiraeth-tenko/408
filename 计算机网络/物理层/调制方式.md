# 调制方式

---

![](数字信号调制.png)

| 名称                | 调制方式                              | 公式           |
| ------------------- | ------------------------------------- | -------------- |
| 辐移键控（ASK）     | 改变**振幅**表示 1 和 0，抗干扰能力差 | 奈氏或香农     |
| 频移键控（FSK）     | 改变**频率**表示 1 和 0，抗干扰能力强 | 奈氏或香农     |
| 相移键控（PSK）     | 改变**相位**表示 1 和 0               | 奈氏或香农     |
| 正交振幅调制（QAM） | 结合 ASK 和 PSK。                     | $R=Blog_2(mn)$ |
