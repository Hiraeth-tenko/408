# 数据报与虚电路

---

|                    | 数据报                                               | 虚电路                               |
| ------------------ | ---------------------------------------------------- | ------------------------------------ |
| 连接               | 不需要建立连接                                       | 需要建立连接                         |
| 目的地址           | 每个分组需要完整的目的地址                           | 建立连接需要；建立连接后使用虚电路号 |
| 路由选择           | 分组独立进行路由                                     | 分组由所属虚电路进行路由             |
| 分组顺序           | 不保证有序                                           | 有序                                 |
| 可靠性             | 由主机保证可靠性                                     | 由网络保证有序                       |
| 网络故障的适应性   | 健壮性较好，故障节点丢失分组，其他分组路由至正常节点 | 经过故障节点的虚电路出错             |
| 差错处理与流量控制 | 尽力而为的交付                                       | 由主机或交换网负责                   |
